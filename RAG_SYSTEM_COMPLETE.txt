  └─ /api/chat (POST endpoint)

  ├─ RAGEngine class

  ├─ generatePrompt() - prompt engineering
  └─ processQuery() - main orchestrator

  ├─ generateLLMResponse() - main function
  ├─ callLLMAPI() - OpenAI integration
  ├─ generateMockResponse() - fallback heuristic
  └─ buildPromptForAPI() - prompt formatting

Curriculum Database (lib/curriculum.ts)
  ├─ CURRICULUM_DATA - 50+ topics
  ├─ NCERTTopic interface
  ├─ searchCurriculum() - search function
  └─ getTopicById() - direct access
